# Component Inventory

## Landing Page (Home)

- **Hero**: Main entry section with GSAP animations.
- **About**: Company background and mission.
- **ServicesIntro/Details**: Showcase of core offerings.
- **WhyPosilenz**: Value proposition and key differentiators.
- **Team**: Introduction to leadership and staff.
- **Insights**: Preview of latest blog posts.
- **Contact**: Inquiry form.
- **Navbar/Footer**: Global navigation elements.

## Blog Feature

- **BlogHero**: Header for the blog section.
- **BlogList**: Grid display of available blog posts.
- **FeaturedPost**: Highlighted post display.
- **BlogDetailClient**: Interactive detail view for articles.
- **SharePopup**: Social sharing interface.

## Careers Feature

- **CareersHero**: Header for job openings.
- **CurrentOpenings**: Filterable list of available roles.
- **ApplicationForm**: Interface for job seekers to apply.
- **WhyJoin**: Company culture and benefits showcase.

## Core UI Components

- **Button**: Standardized button component using Radix Slot.
