# Project Overview

## Purpose

Posilenz.com is a professional website designed to highlight the company’s expertise in data integration, system unification, and workflow optimization. It serves as a hub for services, industry insights (blog), and recruitment (careers).

## Tech Stack Summary

| Category | Technology |
|---|---|
| Framework | Next.js 16.1.1 (App Router) |
| UI | React 19, TailwindCSS 4, GSAP |
| CMS | Keystatic (GitHub Storage) |
| Language | TypeScript 5 |
| Email | Nodemailer, Resend |

## Repository Structure

- **Type**: Monolith
- **Architecture**: Next.js App Router (Layered/Component-based)
- **Primary Language**: TypeScript
